# 🚀 Guide de déploiement — Portail Iqrawartaqi V6 --- ## PARTIE 1 — Firebase (authentification) ### Étape 1 : Créer le projet Firebase 1. Va sur **https://console.firebase.google.com** 2. Clique **Ajouter un projet** → nom : `iqrawartaqi-portail` 3. Désactive Google Analytics (pas nécessaire) → **Créer le projet** ### Étape 2 : Activer l'authentification Email/Mot de passe 1. Dans le menu gauche → **Authentication** → **Commencer** 2. Onglet **Sign-in method** → clique **Email/Mot de passe** → Active → **Enregistrer** ### Étape 3 : Récupérer les clés Firebase 1. Roue dentée ⚙️ en haut à gauche → **Paramètres du projet** 2. Section **Tes applications** → clique l'icône `` (Web) 3. Nom : `portail-web` → **Enregistrer l'application** 4. Tu vois un bloc `firebaseConfig` avec toutes les clés → **copie-les** ### Étape 4 : Coller les clés dans le fichier HTML Ouvre `portail-iqrawartaqi-v6.html` avec un éditeur (Notepad++, VS Code) Cherche ce bloc et remplace : ```js const FB_CONFIG = { apiKey: "AIzaSyBQGwxDavu2P5Xg2q-PJIUVzGdCV_2vFfM", // ← colle ici authDomain: "iqrawartaqi-portail.firebaseapp.com", projectId: "iqrawartaqi-portail", storageBucket: "iqrawartaqi-portail.firebasestorage.app", messagingSenderId: "233822341274", appId: "1:233822341274:web:df04ff19fbddd8e193f723" }; ``` --- ## PARTIE 2 — Créer les comptes utilisateurs Firebase ### Pour chaque admin, prof, élève : 1. Firebase Console → **Authentication** → **Ajouter un utilisateur** 2. Email : l'email de la personne 3. Mot de passe : un mot de passe provisoire (la personne peut le changer) ### ⚠️ IMPORTANT — Les rôles sont gérés dans le portail : Le portail détermine le rôle à la connexion selon l'onglet choisi (Admin / Enseignant / Élève). - Les **admins** : connexion avec l'onglet Admin - Les **profs** : connexion avec l'onglet Enseignant (leur email doit correspondre à leur nom dans la base) - Les **élèves** : connexion avec l'onglet Élève (leur email doit être dans la colonne `email` de la base) --- ## PARTIE 3 — Déploiement sur Vercel ### Étape 1 : Préparer le projet 1. Crée un dossier sur ton PC : `portail-iqrawartaqi/` 2. Place le fichier HTML dedans, renomme-le `index.html` ### Étape 2 : Pousser sur GitHub ```bash git init git add . git commit -m "Portail V6" git remote add origin https://github.com/TON_COMPTE/portail-iqrawartaqi.git git push -u origin main ``` Ou utilise **GitHub Desktop** (interface graphique) si tu n'es pas à l'aise avec Git. ### Étape 3 : Déployer sur Vercel 1. Va sur **https://vercel.com** → Se connecter avec GitHub 2. **New Project** → importe ton repo `portail-iqrawartaqi` 3. Framework : **Other** (pas Next.js, c'est juste un HTML statique) 4. Root Directory : laisser vide → **Deploy** 5. ✅ En 30 secondes, ton portail est en ligne sur `portail-iqrawartaqi.vercel.app` ### Étape 4 : Domaine personnalisé (optionnel) Vercel → ton projet → **Settings** → **Domains** → ajoute `portail.iqrawartaqi.fr` --- ## PARTIE 4 — Ajouter élèves et profs manuellement dans le portail ### Ajouter un élève : 1. Connecte-toi en **Admin** 2. Menu **Élèves** → bouton **+ Ajouter** 3. Remplis : Prénom, Nom, Email, Pays, Matière, Professeur, Type, Durée, Fréquence 4. **Lien groupe Telegram** → colle le lien `https://t.me/+xxx` 5. **Lien Zoom/Meet** → ⚠️ à ajouter dans le champ "Commentaire" pour l'instant (voir section dédiée ci-dessous) ### Ajouter un professeur : 1. Menu **Enseignants** → bouton **+ Ajouter** 2. Remplis : Prénom, Nom, Email (important : doit correspondre au compte Firebase) 3. Programmes enseignés, statut ### Liens Zoom/Meet — Où les mettre ? Le champ n'existe pas encore dans V6. Deux options immédiates : - **Option A** : Colle le lien dans le champ **Commentaire** de l'élève → visible dans sa fiche - **Option B** : Dis-moi et je les ajoute dans la V6.1 en 10 min (champ dédié par élève + affiché dans le tableau de bord élève et enseignant) --- ## PARTIE 5 — Synchronisation Google Sheets Voir le fichier `base-donnees-sheet.gs` fourni séparément. --- ## Récapitulatif des URLs | Service | URL | Gratuit | |---|---|---| | Firebase Auth | console.firebase.google.com | ✅ | | Vercel Hosting | vercel.com | ✅ | | Google Sheets DB | sheets.google.com | ✅ | | Portail en ligne | portail-iqrawartaqi.vercel.app | ✅ |
